    Welcome to my GitHub Page.

    This page was mostly generated automatically by just pressing a button. I'm still a noob at GitHub. I joined GitHub because it appears that a great deal of free software development for the Raspberry Pi is taking place here. I got started when I cloned a repository onto my Rasberry Pi that serves a web app page to control the camera. That app install required Git and also Node to be installed on my Pi. I signed up for a GitHub account and forked the NodeBeginnersBook. In the course of watching how-to videos I discovered that Git was developed by Linus Torvalds, the creator of the Linux operating system. Apparently he wrote it because all the other collaborative software development tools are crap.

    I have managed to 'push' my BerryCam app folder back up to my GitHub repo but I think I made a mistake and uploaded the 'dependency files' that were required after the cloning of the developer's repository.
